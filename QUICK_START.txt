⚡ INSTALLATION (5 minutes):

   1. Extract marketai.zip to your desktop
   
   2. Open Terminal and navigate:
      cd ~/Desktop/marketai
   
   3. Install dependencies:
      npm install
   
   4. Start the app:
      npm run dev
   
   5. Opens automatically at: http://localhost:3000

🔑 AI SETUP (Required):

   1. Get API key: https://console.anthropic.com/
   2. In app: Click Settings → Paste API key → Save
   3. Start chatting with AI!
